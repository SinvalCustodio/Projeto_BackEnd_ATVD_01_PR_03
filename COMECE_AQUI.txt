## 🚀 INICIAR AGORA (3 PASSOS)

### 1️⃣ Instalar dependências
```bash
cd "seu_projeto"
mvn clean install
```

### 2️⃣ Executar aplicação
```bash
mvn spring-boot:run
```

### 3️⃣ Abrir Swagger
```
http://localhost:8080/swagger-ui.html
```

**Pronto!** 🎉 Você terá a API rodando com documentação interativa.

## 📝 Exemplo Rápido

No Swagger, procure por:

**POST /api/usuarios** → Click → Try it out

Preencha com:
```json
{
  "nome": "João Silva",
  "email": "joao@example.com",
  "senha": "123456"
}
```

Click **Execute** → Resultado 201 Created ✅
